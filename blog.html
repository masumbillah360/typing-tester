What is Local Storage and Seassion Storage

Local Storage : LocalStorage - stores data with no expiration date. It Works only inside of Browser. It's capecity up to 10 MB.

Seassion Storage : Data in sessionStorage is cleared when the page session ends or page closed. It's also works only inside of Browser.

Local Scope vs Global Scope

Scope in JavaScript refers to the current context of code, which determines the accessibility of variables to JavaScript. The two types of scope are local and global: Global variables are those declared outside of a block. Local variables are those declared inside of a block

Event Loop in JavaScript

JavaScript has a runtime model based on an event loop, which is responsible for executing the code, collecting and processing events, and executing queued sub-tasks
In a thread, the function at the top of the stack is what's currently running. The event loop is at the bottom of the stack, or at least it's underneath all JavaScript functions, so it only gets to run when all previously running JavaScript functions have returned or thrown.
